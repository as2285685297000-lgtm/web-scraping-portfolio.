

# 🕷️ 网络爬虫作品集

这是一个展示我网络爬虫技能的项目合集，完全使用手机创建和开发。

## 🚀 项目展示

### 1. GitHub用户信息爬虫
- **文件**: `github_scraper.py`
- **功能**: 查询GitHub用户基本信息
- **技术**: Python requests库 + GitHub API
- **运行命令**: `python github_scraper.py`

### 2. 简单网页爬虫  
- **文件**: `simple_scraper.py`
- **功能**: 获取网页标题和基本信息
- **技术**: requests + BeautifulSoup
- **运行命令**: `python simple_scraper.py`

### 3. 测试工具
- **文件**: `test_all.py`
- **功能**: 测试所有爬虫和依赖
- **运行命令**: `python test_all.py`

## 📱 如何在手机上运行

### 方法一：使用Pydroid 3（安卓）
1. 安装Pydroid 3应用
2. 新建文件，粘贴代码
3. 点击运行按钮

### 方法二：使用在线编辑器
1. 访问 [replit.com](https://replit.com)
2. 新建Python项目
3. 粘贴代码并运行
