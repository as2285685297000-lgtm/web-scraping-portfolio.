# 🕷️ 网络爬虫作品集

这是一个展示我网络爬虫技能的项目合集，完全使用手机创建和开发。

## 🚀 项目展示

### 1. GitHub用户信息爬虫
- **文件**: `github_scraper.py`
- **功能**: 查询GitHub用户基本信息
- **技术**: Python requests库 + GitHub API
- **特点**: 
  - 获取用户名、粉丝数、仓库数等
  - 显示用户创建时间
  - 输出头像链接

### 2. 简单网页爬虫
- **文件**: `simple_scraper.py`
- **功能**: 获取网页标题和基本信息
- **技术**: requests + BeautifulSoup
- **特点**: 支持多个网站测试

## 📱 如何在手机上运行

### 方法一：使用Pydroid 3（安卓）
1. 安装Pydroid 3
2. 复制代码到新文件
3. 点击运行按钮

### 方法二：使用在线编辑器
1. 访问 [replit.com](https://replit.com)
2. 新建Python项目
3. 粘贴代码并运行

## 🔧 技术栈

- **Python 3** - 主要编程语言
- **Requests** - HTTP请求库
- **BeautifulSoup4** - HTML解析
- **JSON** - 数据格式处理
- **GitHub API** - 官方接口调用
